# Folder Konten

Simpan file dari Google Drive di sini sesuai kategori:

## Struktur

```
content/
├── tentang/      → File legalitas (NIB, Akta CV), foto lokasi, profil tim
├── layanan/      → Aturan Pembayaran Maklun Bulog, dokumen kerjasama
├── artikel/      → Roadmap Budidaya GAP, Manajemen Risiko, Update Regulasi
└── download/     → Template Surat Perjanjian (PDF/DOC) untuk pengunjung
```

## Gambar untuk Website

Simpan gambar (logo, foto tim, foto lokasi) di:

```
public/images/    → Akan bisa diakses langsung di website sebagai /images/namafile.jpg
```
